BMW USER INTERFACE

An UI with both the usability and the aesthetics.

Reimagining the BMW website interface was a challenge for myself as a designer to preserve the brand's most valued features and create an aesthetically pleasing product that would satisfy both the first time user, by being straightforward and easy to use, and the enthusiast that would like to grasp the boldness of the cars in a digital environment that would please him.
