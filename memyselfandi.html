Case Description

This project is also a product of the Interaction Design course assignments and is meant to incorporate several features for the users, be versatile for both types of devices and include only pieces of clothing from the personal wardrobe, which meant figuring out a style to match those, as well as use certain layout principles.

Specific layout principles

For the overview page I have used the principle of Pagination to showcase all the elements without overwhelming the user and splitting them into several pages. The elements are put in a basic Thumbnail Grid and the logo has the role of an Escape Hatch to take the user from any other page to the overview.

Authentification process

The process of creating an account on the webshop was thoroughly thought out and guides the user through the whole process by hiving them hints and feedback to their action so they never feel overwhelmed or lost. The process ends with a feedback animation, not leaving the user wondering if their account was successfully created or not.
